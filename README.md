🎬 Smart Movie Finder

An AI-powered Movie Recommendation System built using Python, Streamlit, and Machine Learning. The application recommends movies similar to your selected title using a Content-Based Recommendation approach powered by Cosine Similarity and the TMDB API.

Website: https://smart-movie-finder.streamlit.app/

🎥 AI-powered movie recommendations

⚡ Fast recommendation engine with precomputed similarity scores

🧠 How It Works

The application analyzes movie metadata such as genres, cast, crew, keywords, and overview to generate meaningful feature vectors. It then applies Cosine Similarity to identify and recommend the most similar movies. Movie posters and additional details are fetched dynamically using the TMDB API.

📊 Dataset
TMDB 5000 Movies Dataset
TMDB 5000 Credits Dataset
